# LifeOps iOS App

Native SwiftUI app for LifeOps with iOS and Apple Watch support.

## Requirements
- Xcode 15+
- iOS 17+
- watchOS 10+

## Setup
1. Open in Xcode
2. Configure API base URL in settings
3. Build and run

## Features
- Rolling timeline/todo view
- Complete, postpone, skip items
- Streak tracking
- XP rewards
- Apple Watch companion
