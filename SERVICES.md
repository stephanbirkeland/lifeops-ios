## Smart Home Apps (Current State)

| App | Controls | Status |
|-----|----------|--------|
| Apple Home | HomePod, HomeKit devices | Active |
| Google Home | Bathroom speaker | Active |
| Samsung SmartThings | TVs, speakers | Active |
| Plejd | Lighting (Bluetooth) | Active |

### Unification Goal
**Current problem**: Four separate apps to control different devices
**LifeOps goal**: One unified interface that controls everything

Potential solutions:
1. **Home Assistant** - Open source, supports all ecosystems
2. **Custom LifeOps hub** - Built specifically for this setup
3. **Hybrid approach** - Home Assistant backend + custom UI
